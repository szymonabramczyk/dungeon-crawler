\doxysubsection{Detailed Description}
Class for Asset manipulation. 

A class that is responsible for loading, storing and providing easy access to assets. All the assets are stored in their respective maps, using a std\+::string as a key and an unique pointer to the asset. The usage of unique pointers ensures proper assets handling and correct integration with S\+F\+ML way of managing assets.
